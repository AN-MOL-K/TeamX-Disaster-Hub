# TeamX Disaster Connect UI

A modern disaster management and emergency response application built with React, TypeScript, and Tailwind CSS.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:8080`

### Build

```bash
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
